# 键盘快捷键指令

| 键盘快捷键指令     | 定义             |
| ------------------ | ---------------- |
| 上下键             | 查看之前的命令   |
| Ctrl + c           | 终止当前命令执行 |
| Ctrl + Shift + "+" | 终端字体放大     |
| Ctrl + "-"         | 终端字体放小     |
| Ctrl + l           | 清屏             |
| ctrl + x           | 剪切             |
| ctrl + c           | 复制             |
| ctrl + v           | 粘贴             |
