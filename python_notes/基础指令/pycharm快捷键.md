## pycharm常用快捷键

|                        |                    |
| ---------------------- | ------------------ |
|                        |                    |
| home键                 | 移动到本行开头     |
| end键盘                | 移动到本行末尾     |
| ctrl + /               | 注释代码           |
| ctrl +d                | 复制行             |
| 鼠标左键 + alt         | 选择列             |
| shift + alt + 上下箭头 | 移动行             |
| Ctrl + Space           | 智能提示           |
| ctrl + Ait+l           | 不规范格式自动调整 |
